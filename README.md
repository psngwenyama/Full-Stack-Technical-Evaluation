# Enviro365 Junior Software Developer Assessment

## Project Overview

This project is a Spring Boot REST API developed for the Enviro365 Junior Software Developer assessment.

The system allows investors to view their portfolio details, view products linked to portfolios, and submit withdrawal notices based on defined business rules.

## Technologies Used

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- H2 In-Memory Database
- Maven
- JUnit 5
- Mockito
- Swagger / OpenAPI
- React will be used for the frontend
